## Common Issues

### Installation Problems

**Problem**: `ImportError: No module named 'django_revolution'`

**Solution**:
```bash
# Install Django Revolution
pip install django-revolution

# Verify installation
python -c "import django_revolution; print('✅ Installed')"
```

**Problem**: Missing dependencies

**Solution**:
```bash
# Install missing dependencies
pip install django-filter djangorestframework-simplejwt

# Or let Django Revolution install them
python manage.py revolution --install-deps
```

### Configuration Issues

**Problem**: Zone validation failures

**Solution**:
```bash
# Validate zones with detailed output
python manage.py revolution --validate-zones

# Check specific zone
python manage.py revolution --show-urls
```

**Problem**: Schema generation errors

**Solution**:
```bash
# Test schema generation
python manage.py revolution --test-schemas

# Check drf-spectacular installation
pip install drf-spectacular
```

### Performance Issues

**Problem**: Slow generation

**Solution**:
```bash
# Use multithreading
python manage.py revolution --generate --max-workers 16

# Generate specific zones only
python manage.py revolution --zones public
```

**Problem**: High memory usage

**Solution**:
```bash
# Clean before generation
python manage.py revolution --clean --generate

# Reduce worker count
python manage.py revolution --generate --max-workers 4
```

## Self-Diagnosis

### Run Diagnostics

```bash
# Comprehensive system check
python manage.py revolution --status

# Validate environment
python manage.py revolution --validate

# Check zone configuration
python manage.py revolution --validate-zones

# Test schema generation
python manage.py revolution --test-schemas
```

### Debug Mode

```bash
# Enable debug logging
export DJANGO_REVOLUTION_DEBUG=1
python manage.py revolution --debug

# Maximum verbosity
python manage.py revolution --verbosity 3
```

### System Information

When reporting issues, include:

```bash
# System information
python --version
django-admin --version
node --version
npm --version

# Django Revolution version
python -c "import django_revolution; print(django_revolution.__version__)"

# Configuration
python manage.py revolution --status
```

### Development Setup

```bash

git clone https://github.com/markolofsen/django-revolution.git

# Install in development mode
pip install -e .

# Install development dependencies
pip install -r requirements-dev.txt
```
